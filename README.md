# 老杨的匠心片段 · 数据看板

老杨小号矩阵「老杨的匠心片段」的双平台数据看板。纯静态站点（自包含 HTML，无构建、无外部依赖、无 Google Fonts，中国大陆访问正常）。

**入口即驾驶舱**：打开站点根目录 `/` 直接进起号期驾驶舱，点明细表任意一行下钻到单篇详情，不经过额外的导航首页。

## 结构

- `index.html` — **起号期驾驶舱**（双平台横向对比 + 女性受众光谱 + 题材诊断）
- 9 个**下钻页**：小红书单篇 ×5 · 视频号单条 ×4（驾驶舱点行进入，页内可返回驾驶舱）
- 另含（未挂在驾驶舱导航上、可直链访问）：战略看板·麦肯锡版 · 张老师/杨老师战略地图 · 三周战报 · 数据日报

## 部署到 Netlify

### 方式一 · GitHub 自动部署（推荐，改完一推送自动更新）
1. 把本仓库推到 GitHub
2. 登录 [Netlify](https://app.netlify.com) → Add new site → Import an existing project → 选本仓库 → Deploy（构建设置留空，`netlify.toml` 已指定发布目录为根）
3. 之后每次 `git push`，Netlify 自动重新部署

### 方式二 · 直接拖（最快，无需 GitHub）
把整个文件夹拖到 [app.netlify.com/drop](https://app.netlify.com/drop)

## 说明

- 所有页面靠**相对路径**互链，必须整体部署，不可拆分单页。
- 字体已全部改为系统字体栈，无 Google Fonts 依赖。
- 内容里出现的「万里茶道」是视频题材/内容线名称（如"重走万里茶道"），属正常引用；品牌名已统一为「老杨的匠心片段」。
- 如需**访问密码**：Netlify 免费版不含站点密码，可在 `index.html` 注入 JS 密码门实现。
