Prologue

借势 × 合力

一个兼职小团队，凭什么撬动 191.9w 粉的非遗 IP？——靠借势与精准的角色卡位。

借三重势

  • 借老杨的势：抖音 191.9w 粉、苏州明式家具非遗传承人、稀缺的全球行走素材
  • 借水月的对接：经水月（寺庙文创）接入老杨生态，定位为「水月的合作伙伴」
  • 借自己的稀缺：生物制药 + 统计 + 内容三栖，能把内容当可建模对象的人极少

合力卡位（已校准）

  • 张老师（建模 / 方法论 / 小红书执行）× 胡老师 · 兼职双老师团队
  • 不是「老杨方的下游」，不主动碰水月寺庙文创（职业纪律：不这山望那山高）
  • 能扛的：内容质量 + 小红书执行；要补的：种子用户 / 抖音认领 → 钥匙在甲方手里

Gate 1

原力觉醒

痴迷 ∩ 优势 ∩ 回报 —— 三环交叠处，才是可持续的发力点。
优势（已验证 ✓）

理工建模思维迁移：把「凭感觉」的内容变成可建模、可复现。五层评分模型（视频像蛋白质折叠，剪辑错序=misfolding 失活）· 信息酶 7 酶选题模型 · 用概率统计量化双平台数据。

痴迷 ⊙ 待你盘点

vault 里能坐实的是「持续做方法论建模」，但你究竟痴迷的是内容本身，还是『把软问题硬建模』这件事——逐字稿未覆盖，需你自评。这决定方法论该往哪个方向长。

回报

短期＝给甲方过程性成效（不必立刻 GMV）；长期＝方法论可复制 / 产品化。但具体回报形态（咨询？课程？合伙分成？）⊙ 待你定。

2
Gate 2

品类独创

不在红海里卷，开一个只有你能定义的新品类。

独创品类：「内容工程学」

  • 用酶 / 折叠 / 概率的硬建模做内容方法论——一个目前几乎无人占据的交叉品类
  • 区别于「经验派运营」（凭手感、不可复现）
  • 区别于「精致美学派」（红海、与真实感冲突）
  • 主张：内容像酶——可设计、可催化、可复现；选题=信息酶，质检=五层折叠

甜用户：谁为这套方法论买单？ ⊙ vault 未覆盖

  • 候选 A：想做 IP 但不懂方法的非遗匠人 / 文化创业者（老杨是第一个样品）
  • 候选 B：缺方法论沉淀的中小内容团队 / MCN
  • 候选 C：高校 / 培训场景（把方法论变课程）
  • 需你拍板：甜用户定义不清，品类就只是自嗨，无法变现

已成立的升维逻辑

  • 账号是样品，方法论才是产品——三个月后交出的是「别人拿着也能跑的内容运营方法」
  • 双 IP 双酶接力：非遗线建认同（E3）→ 茶道线做转化（E4/E5），是可复用的链路设计

升维缺口 ⊙ 待你盘点

  • 获客结构：现在靠水月转介，方法论要卖给甜用户走什么渠道？
  • 利润结构：兼职无明确商业模式——咨询费 / 课程 / 分成 / 持股？
  • 交付物形态：方法论以什么打包（手册 / SOP / SaaS 看板 / 陪跑）？

Gate 4

壁垒锁定

护城河在哪？哪些是真壁垒，哪些是会被填平的浅沟。

护城河候选（真壁垒）

  • 跨域稀缺：生物制药科研 + 统计 + 内容操盘，三者叠加的人极少——别人难复制的不是单项，是组合
  • 方法论 IP：五层模型 / 信息酶 / 双酶接力是自创、可固化、可署名的资产
  • 真实数据资产：老杨项目积累的双平台实测（点赞率 / 完播 / 受众画像），是方法论的燃料

壁垒风险（浅沟预警）

  • 单人承载过重——与老杨同病；兼职精力是硬约束
  • 方法论尚 N=1——只在老杨一个案例上验证，未被第三方复现前不算护城河
  • 角色依附——目前势能借自老杨/水月，自有甜用户未建立前壁垒是「租来的」
  • 你的护城河自评⊙ 待你盘点：上述哪条你最想加固？
